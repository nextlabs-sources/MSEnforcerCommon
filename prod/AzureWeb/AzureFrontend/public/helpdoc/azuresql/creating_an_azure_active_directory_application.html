Creating an Azure Active Directory application

This section describes the steps that are required to create an Azure Active Directory application.

  1. In your browser, type:

    https://azure.microsoft.com/en-gb/features/azure-portal/

  2. Log in to your Azure account using an account that has administrator privileges.
    Note: If your account gives you access to more than one tenant, click your account in the top-right corner, and set your portal session to the target Azure AD tenant.
  3. On the left navigation pane, click Azure Active Directory.
  4. Click App registrations.
  5. Click New application registration.
  6. In the Name box, specify a meaningful name for the application.
  7. From the Application type drop-down list, select Web app / API.
  8. In the Sign-on URL box, type the following URL.
    https://cloudproxy.azurewebsites.net/callback/azure
  9. Click Create.
